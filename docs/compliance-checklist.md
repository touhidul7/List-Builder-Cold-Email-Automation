# Compliance Checklist

- [ ] No confidential deal info before NDA.
- [ ] Clear opt-out line.
- [ ] No misleading claims.
- [ ] No automatic campaign launch.
- [ ] Valid emails only by default.
- [ ] Catch-all requires manual review.
- [ ] Canadian CASL review before outreach.
- [ ] Investor campaigns require human approval.
- [ ] Paid actions require approval.
- [ ] Store only necessary business-contact data.
- [ ] Maintain suppression and do-not-contact handling.
